Portfólio/proposta comercial da Emmily Dias, feito em [Next.js](https://nextjs.org) + Tailwind.

## Getting Started

Run the dev server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Edite a página em `src/app/page.tsx` e o conteúdo em `src/lib/content.ts`.

## Deploy (Vercel)

Opção 1 (recomendado): conecte o repositório no painel da Vercel e faça o deploy.

Opção 2 (CLI): faça login e publique:

```bash
npx vercel login
npx vercel --prod
```
